## Technologies Used

- React

## Running Instructions

1. Clone the project:

```
git clone https://github.com/matheusgzancanela/supermarket-list-frontend.git
```

2. Install the dependencies:

```
cd supermarket-list-frontend && npm install
```

3. Run the project:

```
npm start
```
